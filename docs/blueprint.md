# **App Name**: Portfolio Pro

## Core Features:

- Interactive Resume: Dynamic display of work experience with timeline and bullet-point achievements.
- Project Showcase: Highlight featured projects with descriptions, tech stacks, live demos, and GitHub links.
- Skills Visualization: Visually engaging representation of technical skills and proficiencies.
- AI-Powered Portfolio Review Tool: An AI-powered tool provides personalized feedback to portfolio viewers, improving their engagement by addressing specific questions regarding the resume and projects. 
- Contact Form: Simple contact form or links to email, LinkedIn, and GitHub.

## Style Guidelines:

- Primary color: Blue (#13A4EC), representing trust and professionalism.
- Background color: Light gray (#F6F7F8) for a clean and modern feel in light mode, dark gray (#101C22) in dark mode.
- Accent color: Slightly darker blue (#0D86C2) for interactive elements, providing subtle contrast.
- Font: 'Inter', a sans-serif font, known for its readability and modern appearance, used for both headings and body text.
- Use Material Design icons for a consistent and recognizable visual language.
- Clean and minimalist layout with ample whitespace to ensure readability and focus on content.
- Subtle animations on hover and transitions to enhance user experience without being distracting.
